# Workspace Templates

These files are Vicode's first-party bootstrap templates for workspace context.

They are intended to be:

- reviewed by humans
- filled from repo inspection and bootstrap questionnaire answers
- written into trusted project workspaces only after user approval

Placeholder conventions:

- `{{REPO_*}}` values come from repo inspection
- `{{USER_*}}` values come from the bootstrap questionnaire
- `{{APP_*}}` values come from Vicode defaults
- optional sections may be omitted when no value exists

These templates are informed by:

- OpenClaw workspace and bootstrapping docs
- Claude Code memory guidance
- OpenAI and Anthropic prompt-quality guidance
